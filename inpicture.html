Features

LiveEdu is a web application that takes charge of grade management, schooling and discipline in any establishment. Here are some of the features detailed below.

List of evaluations.

This list is paginated, the user has the possibility to search for an evaluation according to the class, the session and the course evaluated.

Illustration

New evaluation.

The evaluation collects student grades in one session, on a discipline. Each teacher can create/modify/delete evaluations with the agreement of the administration. When editing an evaluation, the lists of students in the class are listed in alphabetical order, with the evaluations automatically assigned by default.

Illustration

Course assignment

The responsibility for teaching a course in a class is defined by the administrator each new school year must be done in a sequence. The form is designed so that it is impossible for the same subject to be assigned to different teachers in the same year.

Illustration

Generation of report cards.

Sequential, quarterly or annual report cards can be generated by class or by student. They consist of an A4 format document.

Demonstration

Registration.

Enrolling a student in a class in one school year must be done. The form is designed so that it is impossible for the same student to be enrolled in different classes in the same year.

Demonstration

Session.

The session determines the period of the quarter for which the application operates. Each assessment should be done in a sequence. The administrator is responsible for creating/modifying/deleting them.

About US
